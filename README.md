# Ecmascript 6+

This project contains all Ecmascript features relased since ES6 (2015)


## ES6 (2015)
- Default params
- Template literals
- Destructuring
- Spread operator
- Let & const
- Object property enhancements
- Arrow functions
- Promises
- Classes
- Modules
- Generators

## ES7 (2016)
- Includes method
- Potentiation operator

## ES8 (2017)
- Object.entries
- Object.values
- Padding
- Async & Await
